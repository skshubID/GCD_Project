#COURSERA
##Getting and Cleaning Data

**Course Project**
*Create a 'Tidy Dataset' using raw data collected from the accelerometers from the Samsung Galaxy S smartphone.*

**Packages Used**
'dplyr' is used in the project.
To install: > install.packages("dplyr")
To load: > library(dplyr)

**R Script (run_analysis.R)**
The script reads in the data files from provided link and writes out a tidy dataset (tidy_data.txt)

**Directory**
Current directory contains the script file and tidy data set will be written to the current directory.
Current directory has a subdirectory (./data), that contains all data as downloaded.
